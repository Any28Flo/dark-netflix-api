Welcome to Dark Netflix API. (Beta)


QUOTES

  • /api/v1/quote - for entire Quotes data in JSON.
  • /api/v1/quote/id - for Quote data by id in JSON.
  • /api/v1/quote/random - for random (single) Quotes data in JSON.
  • /api/v1/quote?author=Name+Surname - for entire Quotes data by author in JSON.
  • /api/v1/quote/random?author=Name+Surname - for random (single) Quotes data by author in JSON.

CHARACTERS

  • /api/v1/character - for entire Character data in JSON.
  • /api/v1/character/id - for Character data by id in JSON.
  • /api/v1/character/random - for random (single) Character data in JSON.
  • /api/v1/character?name=Name+Surname - for Character data by name in JSON.

EPISODES

  • /api/v1/episode - for entire Episode data in JSON.
  • /api/v1/episode/id - for Episode data by id in JSON.
  • /api/v1/episode/random - for random (single) Episode data in JSON.
  • /api/v1/episode?season=Number - for entire Episode data by season in JSON.
  • /api/v1/episode/random?season=Number - for random (single) Episode data by season in JSON.
